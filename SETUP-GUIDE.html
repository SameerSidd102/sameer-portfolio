SETUP GUIDE

How to get your portfolio live with a free admin panel — no code, ever. Should take about 20 minutes.

STEP 1

Create a free GitHub account

GitHub stores your website files (like a USB stick in the cloud). You'll never actually use GitHub again after setup — it just sits in the background.

  • Go to github.com
  • Click Sign up
  • Use any email — it's completely free
  • Verify your email when they send you a code
→ Create GitHub account
STEP 2

Create a free Netlify account

Netlify hosts your website and gives you the admin login panel. Also completely free.

  • Go to netlify.com
  • Click Sign up
  • Choose "Sign up with GitHub" — this links the two accounts automatically
  • Authorise the connection when it asks
→ Create Netlify account
STEP 3

Upload your portfolio folder to GitHub

You downloaded a folder called sameer-portfolio with all your files inside. Now you upload it to GitHub.

  • In GitHub, click the + button (top right) → New repository
  • Name it: sameer-portfolio
  • Make sure it's set to Public
  • Click Create repository
  • On the next screen, click "uploading an existing file"
  • Drag your entire sameer-portfolio folder contents into the box
  • Click Commit changes
💡 Drag the contents of the folder, not the folder itself — so GitHub sees index.html, admin/, _data/, etc. at the top level.
STEP 4

Deploy to Netlify

This makes your site live on the internet with a real URL.

  • In Netlify, click Add new site → Import an existing project
  • Choose GitHub
  • Find and select your sameer-portfolio repository
  • Leave all settings as they are — just click Deploy site
  • Wait about 60 seconds — Netlify gives you a URL like sparkling-unicorn-123.netlify.app
🎉 Your portfolio is now live! You can rename the URL to something like sameersaeed.netlify.app in Site settings → Domain management.
STEP 5

Turn on the admin panel (Identity + Git Gateway)

This is what gives you the WordPress-style login. Two toggles to turn on in Netlify.

  • In Netlify, go to your site → Site configuration → Identity
  • Click Enable Identity
  • Scroll down to Services → click Enable Git Gateway
  • Go back to Identity → Invite users
  • Invite your own email address
  • Check your inbox — click the confirmation link
  • Set your password
⚠️ Important: Under Identity → Registration, set to Invite only — so nobody else can log into your admin.
STEP 6

Log into your admin panel

You're done! Your admin panel lives at:

https://YOURSITE.netlify.app/admin

  • Go to that URL
  • Log in with the email and password you just set
  • You'll see: Homepage, Case Studies, Experience, Testimonials, Creative
  • Click any section to edit — change text, upload images, add case studies
  • Hit Publish — your site updates within 30 seconds
📱 Bookmark /admin on your phone too — you can update your site from anywhere.
WHAT YOU CAN EDIT

Everything — here's what's in the admin

  • Homepage — your name, role, pitch, all 4 stats, about text, email, LinkedIn, Instagram
  • Case Studies — add/edit/delete any case study. Title, problem, what you did, 3 metrics, tags, colour
  • Experience / CV — add/reorder/edit every job on the timeline
  • Testimonials — swap in your real LinkedIn recommendations
  • Creative → Videos — paste YouTube/Vimeo links or upload files
  • Creative → Designs — upload images directly from your device

YOU'RE LIVE.

Free forever. No monthly fees. Edit from your phone. No code required.
